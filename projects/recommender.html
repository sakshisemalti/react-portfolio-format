Face Recognition

  • Tech Stack: Python, Pandas, NumPy, Scikit-learn, Streamlit, Pickle, Seaborn
  • Github URL: Project Link

Implemented a content-based movie recommendation system to suggest movies based on user-selected titles Processed and cleaned movie datasets, extracting features such as title, genres, cast, and director Created a combined feature column by merging relevant movie attributes for better similarity computation Applied CountVectorizer to convert textual movie features into numerical vectors Computed cosine similarity between movies to identify the most similar titles Designed a recommendation function that returns top-N similar movies for a given input movie Developed an interactive web application using Streamlit to allow users to search for movies and get real-time recommendations Used Pickle to save the similarity matrix and preprocessed data for fast loading and efficient predictions. Visualized movie similarity patterns and popular genres using Matplotlib/Seaborn.
